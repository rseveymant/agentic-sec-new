# v1 archive

This directory contains the original 859-line single-file prototype that informed the v2 architecture. It is kept here as a reference snapshot, not as code that v2 imports or runs against.

The new architecture lives at the repo root. See the top-level README for run instructions.

Original files:

- `agentic_security_demo.py` — single-file prototype: simulator + CLI + minimal HTTP UI all in one place.
- `README.md` — original v1 explainer.

Per ADR-7 in `aisdlc-docs/inception/1-design.md`, v1 is preserved (not deleted) so the simulation's prior shape stays inspectable. Do not modify these files.
